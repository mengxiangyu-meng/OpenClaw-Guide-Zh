# 06-技能系统

> 50+ 内置技能 + 自定义技能开发

## 什么是技能？

技能（Skill）是 OpenClaw 的核心扩展机制，让 AI 助手具备各种专业能力。比如查天气、搜资料、管文件等，都是通过技能实现的。

## 内置技能一览

OpenClaw 预装了大量常用技能：

| 类别 | 技能 | 说明 |
|------|------|------|
| 🔍 搜索 | web_search | 搜索互联网 |
| 📄 读取 | web_fetch | 获取网页内容 |
| 📊 分析 | pdf | 分析 PDF 文档 |
| 🖼️ 图像 | image | 分析图片内容 |
| 📝 写作 | feishu_doc | 飞书文档操作 |
| ☁️ 存储 | feishu_drive | 飞书云盘操作 |
| 📚 知识 | feishu_wiki | 飞书知识库 |
| 📊 表格 | feishu_bitable | 飞书多维表格 |
| 🌤️ 天气 | weather | 查询天气 |

## 使用技能

技能会在 AI 判断需要时自动调用。比如你说"帮我搜一下最新的 AI 新闻"，AI 会自动使用 web_search 技能。

## 自定义技能

### 基本结构

创建一个技能需要以下文件：

```
my-skill/
├── SKILL.md          # 技能定义文件
└── references/       # 参考资料（可选）
```

### SKILL.md 格式

```yaml
---
name: my-skill
description: 我的自定义技能，用于...
---

# 技能名称

## 概述
简要说明这个技能做什么

## 使用场景
- 场景1
- 场景2
```

## 技能开发最佳实践

### 1. 描述要准确

技能描述应该清楚说明：
- 技能做什么
- 什么场景下使用
- 有什么限制

### 2. 提供示例

在 SKILL.md 中给出具体的使用示例。

### 3. 组合使用

多个简单技能比一个复杂技能更好。
