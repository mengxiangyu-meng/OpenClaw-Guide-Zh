# OpenClaw 安装部署

> 在 macOS、Linux、Windows 上安装 OpenClaw

---

## 2.1 环境要求

| 项目 | 要求 |
|------|------|
| Node.js | 22.12.0+ 或 22 LTS (22.16+) |
| 操作系统 | macOS 10.15+ / Linux / Windows (WSL2) |
| 存储空间 | 约 500MB |
| 网络 | 需要访问 GitHub 等外网 |

> ⚠️ **注意**：不支持原生 Windows，需要通过 WSL2 或 Docker 运行

---

## 2.2 安装步骤

### 2.2.1 macOS / Linux 安装

```bash
# 1. 确保已安装 Node.js 22+
node --version

# 2. 安装 OpenClaw
npm install -g openclaw@latest

# 3. 验证安装
openclaw --version
```

### 2.2.2 Windows (WSL2) 安装

```bash
# 1. 打开 WSL2 终端（Ubuntu 等）

# 2. 安装 Node.js 22
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt-get install -y nodejs

# 3. 安装 OpenClaw
npm install -g openclaw@latest

# 4. 验证
openclaw --version
```

### 2.2.3 Docker 安装

```bash
# 1. 拉取镜像
docker pull openclaw/openclaw:latest

# 2. 运行容器
docker run -d \
  --name openclaw \
  -p 18789:18789 \
  -v ~/.openclaw:/home/node/.openclaw \
  openclaw/openclaw:latest
```

---

## 2.3 初始化配置

```bash
# 运行初始化向导
openclaw onboard --install-daemon
```

这个命令会：
1. 检查环境配置
2. 引导你完成基本设置
3. 安装系统服务（可选）

---

## 2.4 常见问题

### Q1: npm 安装失败？

```bash
# 方案1：使用淘宝镜像
npm install -g openclaw@latest --registry=https://registry.npmmirror.com

# 方案2：检查 Node.js 版本
node --version  # 确保是 22+
```

### Q2: 权限报错？

```bash
# macOS/Linux 可能需要 sudo
sudo npm install -g openclaw@latest
```

### Q3: Windows 怎么安装？

推荐使用 WSL2（Windows Subsystem for Linux），或者用 Docker 方式安装。

---

## 2.5 验证安装

安装完成后，运行：

```bash
openclaw --version
```

应该看到类似输出：
```
OpenClaw v2026.2.x
```
