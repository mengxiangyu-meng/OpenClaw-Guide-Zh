## 5.2 登录消息平台

### 5.2.1 Telegram 登录

```bash
# 1. 启动登录
openclaw channels login telegram

# 2. 按提示打开 Telegram
# 搜索 @BotFather 创建机器人，获取 Token

# 3. 配置 Token
openclaw channels config telegram --token "your-bot-token"
```

**详细步骤：**
1. 在 Telegram 搜索 @BotFather
2. 发送 /newbot 创建新机器人
3. 获取 Bot Token
4. 把 Token 配置到 OpenClaw

### 5.2.2 Discord 登录

```bash
# 1. 启动登录
openclaw channels login discord

# 2. 创建 Discord 应用
# https://discord.com/developers/applications

# 3. 获取 Bot Token 和 Channel ID

# 4. 配置
openclaw channels config discord \
  --token "your-discord-token" \
  --channel "your-channel-id"
```

### 5.2.3 飞书登录

```bash
# 1. 启动登录
openclaw channels login feishu

# 2. 创建飞书应用
# https://open.feishu.cn/

# 3. 配置 App ID 和 Secret
openclaw channels config feishu \
  --app-id "your-app-id" \
  --app-secret "your-app-secret"
```

## 5.3 配置示例

### 5.3.1 Telegram 单人聊天

```json
{
  "channels": {
    "telegram": {
      "enabled": true,
      "token": "your-bot-token"
    }
  }
}
```

### 5.3.2 Discord 群聊

```json
{
  "channels": {
    "discord": {
      "enabled": true,
      "token": "your-bot-token",
      "channels": {
        "allowed": ["123456789"]
      }
    }
  }
}
```

### 5.3.3 飞书企业

```json
{
  "channels": {
    "feishu": {
      "enabled": true,
      "appId": "your-app-id",
      "appSecret": "your-app-secret"
    }
  }
}
```

## 5.4 权限设置

### 5.4.1 允许特定用户

```json
{
  "channels": {
    "telegram": {
      "allowFrom": ["user-id-1", "user-id-2"]
    }
  }
}
```

### 5.4.2 群聊需要 @mention

```json
{
  "channels": {
    "discord": {
      "groups": {
        "*": {
          "requireMention": true
        }
      }
    }
  }
}
```

## 5.6 常见问题

### Q1: Telegram 收不到消息？

1. 检查 Bot Token 是否正确
2. 确保已和 @BotFather 开启隐私模式（可选）
3. 用 `/start` 激活机器人

### Q2: Discord 机器人不响应？

1. 检查是否正确添加了 intents
2. 确保机器人已加入频道
3. 检查权限设置

### Q3: 飞书收不到消息？

1. 检查 App ID/Secret 是否正确
2. 确认已开启机器人能力
3. 检查事件订阅地址
