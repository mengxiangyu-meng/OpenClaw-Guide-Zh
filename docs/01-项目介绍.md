# OpenClaw 项目介绍

> 了解 OpenClaw 是什么、它能做什么、以及为什么你需要它

---

## 1.1 OpenClaw 是什么？

**OpenClaw** 是一个开源的 AI 助手网关框架，通俗点说：

> 它让你的 AI 助手可以在各种聊天软件里跟你随时随地聊天

想象这个场景：
- 早上在 **微信** 问 AI 今天的天气怎么样
- 上午在 **Telegram** 让 AI 帮你写代码
- 中午在 **Discord** 群聊里用 AI 解答问题
- 下午在 **飞书** 让 AI 帮你整理会议要点

**一个 OpenClaw，全部搞定。**

---

## 1.2 核心特性

| 特性 | 说明 |
|------|------|
| 🏠 自托管 | 数据存在你自己电脑上，安全可控 |
| 🌐 多平台 | 同时支持微信/Telegram/Discord/飞书/iMessage |
| 🤖 Agent 原生 | 专为 AI 助手设计，支持会话、记忆、工具调用 |
| 📱 移动端 | 支持 iOS/Android 节点，拍照、语音都能用 |
| 🎨 Web 控制台 | 浏览器直接管理所有会话和配置 |
| 🆓 开源免费 | MIT 协议，社区活跃 |

---

## 1.3 工作原理

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  微信/Telegram/   │     │             │     │   AI 模型   │
│  Discord/飞书    │────▶│  OpenClaw   │────▶│ (Claude/GPT)│
│  (消息平台)      │     │   Gateway    │     │             │
└─────────────┘     └─────────────┘     └─────────────┘
       ▲                   │
       │                   ▼
       │            ┌─────────────┐
       └────────────│  Web 控制台  │
                    └─────────────┘
```

**核心组件：**

1. **Gateway（网关）** - 核心进程，负责连接消息平台和 AI 模型
2. **消息平台** - 你常用的聊天软件（微信、Telegram等）
3. **AI 模型** - 提供智能能力（Claude、GPT、Ollama等）
4. **Web 控制台** - 浏览器管理界面
