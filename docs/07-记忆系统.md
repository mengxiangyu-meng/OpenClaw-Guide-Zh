# 07-记忆系统

> AI 如何记住你的偏好和上下文

## 为什么需要记忆？

想象一下：
- 你告诉 AI 你叫"孟孟"，下次它应该记住
- 你之前让 AI 帮你总结了某篇文章，再次提到时它应该知道
- 你的工作习惯、偏好设置，AI 应该长期记住

这就是记忆系统的作用。

## 记忆类型

### 1. 会话记忆

当前对话的上下文，AI 自动记住你说的话。

### 2. 日志记忆

记录每天发生的重要事情，便于回顾。

文件位置：`memory/YYYY-MM-DD.md`

### 3. 长期记忆

持久化的重要信息，跨会话记忆。

文件位置：`MEMORY.md`

```markdown
# 孟孟的偏好
- 喜欢简洁直接的沟通方式
- 学习 GitHub 中
- 时区：UTC
```

## 配置记忆系统

### 配置文件

在 `~/.openclaw/config.yaml` 中配置：

```yaml
memory:
  # 记忆保留天数
  retentionDays: 30
  
  # 每日记忆文件位置
  dailyPath: "./memory"
  
  # 长期记忆文件
  longTermPath: "./MEMORY.md"
```

## 记忆最佳实践

### 1. 及时记录

重要的约定、偏好、决定，立刻写下来。

### 2. 定期整理

定期回顾 daily 文件，提取重要内容到 MEMORY.md。

### 3. 分清主次

- **短期记忆**：daily 文件，放详细日志
- **长期记忆**：MEMORY.md，放精选内容
