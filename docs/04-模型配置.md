# OpenClaw 模型配置

> 配置你喜欢的 AI 模型（Claude/GPT/Ollama 等 29 个提供商）

---

## 4.1 支持的模型提供商

OpenClaw 支持超多的 AI 模型提供商：

| 提供商 | 模型示例 | 需要 |
|--------|---------|------|
| Anthropic | Claude Sonnet/Haiku | API Key |
| OpenAI | GPT-4o / GPT-4o mini | API Key |
| Google | Gemini 1.5 Pro/Flash | API Key |
| Ollama | Llama3 / Mistral / Qwen | 本地运行 |
| 腾讯云 | Hunyuan | API Key |
| 阿里云 | Qwen / Tongyi | API Key |
| ... | 29+ 提供商 | ... |

---

## 4.2 配置文件位置

配置文件在：`~/.openclaw/openclaw.json`

**创建/编辑配置文件：**

```bash
# 方法1：用编辑器打开
nano ~/.openclaw/openclaw.json

# 方法2：用 openclaw 命令
openclaw config edit
```

---

## 4.3 主流模型配置

### 4.3.1 Anthropic Claude

```json
{
  "model": {
    "provider": "anthropic",
    "model": "claude-sonnet-4-20250514",
    "apiKey": "sk-ant-your-anthropic-key"
  }
}
```

**常用 Claude 模型：**
- `claude-sonnet-4-20250514` - 性价比高，推荐！
- `claude-opus-4-20250514` - 最强能力
- `claude-haiku-3-20240307` - 速度快，便宜

### 4.3.2 OpenAI GPT

```json
{
  "model": {
    "provider": "openai",
    "model": "gpt-4o",
    "apiKey": "sk-your-openai-key"
  }
}
```

**常用 GPT 模型：**
- `gpt-4o` - 最新最强
- `gpt-4o-mini` - 便宜快速
- `gpt-4-turbo` - 性价比不错

### 4.3.3 Ollama 本地模型

```json
{
  "model": {
    "provider": "ollama",
    "model": "llama3"
  }
}
```

> 💡 先安装 Ollama 并下载模型：`ollama pull llama3`

### 4.3.4 腾讯 Hunyuan（国内可用）

```json
{
  "model": {
    "provider": "tencent",
    "model": "hunyuan-200k",
    "apiKey": "your-tencent-key"
  }
}
```

## 4.5 中转 API

如果官方 API 访问困难，可以使用中转服务：

```json
{
  "model": {
    "provider": "anthropic",
    "model": "claude-sonnet-4-20250514",
    "baseUrl": "https://api.example.com/v1",
    "apiKey": "your-proxy-key"
  }
}
```

---

## 4.6 多模型配置

可以配置多个模型，在不同场景切换：

```json
{
  "models": {
    "default": {
      "provider": "anthropic",
      "model": "claude-sonnet-4-20250514"
    },
    "fast": {
      "provider": "openai", 
      "model": "gpt-4o-mini"
    },
    "local": {
      "provider": "ollama",
      "model": "llama3"
    }
  }
}
```

---

## 4.7 验证配置

修改配置后，重启 Gateway：

```bash
# 停止
Ctrl+C

# 重新启动
openclaw gateway --port 18789
```

在控制台发送消息测试是否正常工作。
