# OpenClaw 快速开始

> 5 分钟跑起你的第一个 AI 对话

---

## 3.1 启动 Gateway

安装完成后，一行命令启动：

```bash
openclaw gateway --port 18789
```

启动成功会看到类似输出：
```
🚀 OpenClaw Gateway 启动中...
📡 监听端口: 18789
🌐 控制台地址: http://127.0.0.1:18789
```

---

## 3.2 打开控制台

打开浏览器访问：**http://127.0.0.1:18789**

你会看到 OpenClaw 控制台界面，可以在里面：
- 💬 和 AI 聊天
- ⚙️ 配置设置
- 📱 管理已连接的消息平台

---

## 3.3 配置 AI 模型（可选）

默认会使用内置的 Pi 模型。如果你想用其他模型：

### 3.3.1 配置 Claude

编辑配置文件 `~/.openclaw/openclaw.json`：

```json
{
  "model": {
    "provider": "anthropic",
    "model": "claude-sonnet-4-20250514",
    "apiKey": "sk-ant-your-key-here"
  }
}
```

### 3.3.2 配置 GPT

```json
{
  "model": {
    "provider": "openai",
    "model": "gpt-4o",
    "apiKey": "sk-your-key-here"
  }
}
```

### 3.3.3 配置 Ollama（本地模型）

```json
{
  "model": {
    "provider": "ollama",
    "model": "llama3"
  }
}
```

---

## 3.4 连接消息平台

### 3.4.1 Telegram 登录

```bash
openclaw channels login
```

选择 Telegram，按提示完成登录。

### 3.4.2 Discord 登录

```bash
openclaw channels login discord
```

### 3.4.3 飞书登录

```bash
openclaw channels login feishu
```

---

## 3.5 开始聊天

连接好消息平台后，就可以开始聊天了！

| 平台 | 怎么开始 |
|------|----------|
| Telegram | 给机器人发消息 |
| Discord | 在频道 @机器人 |
| 飞书 | 私信机器人 |

---

## 3.6 快速命令一览

| 命令 | 说明 |
|------|------|
| `openclaw gateway` | 启动网关 |
| `openclaw channels login` | 登录消息平台 |
| `openclaw channels list` | 查看已连接的平台 |
| `openclaw status` | 查看状态 |
